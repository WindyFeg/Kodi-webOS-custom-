- Variable name: LG_WEBOS_TV_SDK_HOME
- Variable value: "..\KodiWebOS\webOS_TV_SDK" -> \CLI

2. Install extension webOS TV

3. Install Developer Mode on TV 
- Turn Dev Mode + Key Server

4. Set-up Device
- name, Ip TV, Port: 22
